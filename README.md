## What
Standard GemSet for development.

## Determination Criteria
 * Usefull
 * Less harmful
 * Easy Testable

## Install

In your `Gemfile` add:

```ruby
gem 'developmentkit', group: :development
```

Then run

```sh
$ bundle install
```

Now you're good to go.

## If conflict,
```sh
$ rake developmentkit:update
```
